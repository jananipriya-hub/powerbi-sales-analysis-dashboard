# UCI Sales Dashboard - Power BI

This project analyzes the UCI Sales dataset using Power BI.

## Features
- Interactive slicers
- Sales trend analysis
- Region-wise performance
- Product category insights
- KPI cards for revenue and sales

## Tools Used
- Power BI
- Excel
- Data Cleaning
- Data Modeling
